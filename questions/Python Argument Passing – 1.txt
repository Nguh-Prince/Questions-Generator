1. What is the type of each element in sys.argv?
a) set
b) list
c) tuple
d) string


Answer: d
Explanation: It is a list of strings.



2. What is the length of sys.argv?
a) number of arguments
b) number of arguments + 1
c) number of arguments – 1
d) none of the mentioned


Answer: b
Explanation: The first argument is the name of the program itself. Therefore the length of sys.argv is one more than the number arguments.

(Python Argument Passing – 1) 8. What will be the output of the following Python code?

def foo():
    total += 1
    return total
total = 0
print(foo())

a) 0
b) 1
c) error
d) none of the mentioned


Answer: c
Explanation: It is not possible to change the value of a global variable without explicitly specifying it.



(Python Argument Passing – 1) 9. What will be the output of the following Python code?

def foo(x):

    return id(x)
q = ['abc', 'def']
print(id(q) == foo(q))

a) True
b) False
c) None
d) Error


Answer: b
Explanation: A new object is created in the function.



(Python Argument Passing – 1) 10. What will be the output of the following Python code?

def foo(i, x=[]):
    x.append(i)
    return x
for i in range(3):
    print(foo(i))

a) [0] [1] [2] 
b) [0] [0, 1] [0, 1, 2] 
c) [1] [2] [3] 
d) [1] [1, 2] [1, 2, 3] 


Answer: b
Explanation: When a list is a default value, the same list will be reused.
