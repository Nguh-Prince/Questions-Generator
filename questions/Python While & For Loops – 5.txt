(Python While & For Loops – 5) 1. What will be the output of the following Python code?

for i in range(2.0):
    print(i)

a) 0.0 1.0
b) 0 1
c) error
d) none of the mentioned


Answer: c
Explanation: Object of type float cannot be interpreted as an integer.



(Python While & For Loops – 5) 2. What will be the output of the following Python code?

for i in range(int(2.0)):
    print(i)

a) 0.0 1.0
b) 0 1
c) error
d) none of the mentioned


Answer: b
Explanation: range(int(2.0)) is the same as range(2).

(Python While & For Loops – 5) 5. What will be the output of the following Python code snippet?

for i in [1, 2, 3, 4][::-1]:
    print(i, end=' ')

a) 1 2 3 4
b) 4 3 2 1
c) error
d) none of the mentioned


Answer: b
Explanation: [::-1] reverses the list.

(Python While & For Loops – 5) 9. What will be the output of the following Python code snippet?
