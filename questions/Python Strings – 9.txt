(Python Strings – 9) 1. What will be the output of the following Python code?

print('{0:.2}'.format(1/3))

a) 0.333333
b) 0.33
c) 0.333333:.2
d) Error


Answer: b
Explanation: .2 specifies the precision.



(Python Strings – 9) 2. What will be the output of the following Python code?

print('{0:.2%}'.format(1/3))

a) 0.33
b) 0.33%
c) 33.33%
d) 33%


Answer: c
Explanation: The symbol % is used to represent the result of an expression as a percentage.
